# TaskNote Academy 📝✨

Projeto desenvolvido para a **Web Academy UFAC**, com o objetivo de criar uma aplicação web simples e moderna para gerenciamento de tarefas, inspirada no visual do **Google Keep**.

---

## 📌 Sobre o Projeto

O **TaskNote Academy** é uma aplicação de lista de tarefas (To-Do List) onde o usuário pode:

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Alternar entre **modo claro** e **modo escuro**

O foco inicial do projeto é o **Front-End**, utilizando HTML, CSS e JavaScript.

---

## 🎯 Objetivo

Criar uma interface amigável e responsiva, aplicando conceitos aprendidos na Web Academy UFAC, como:

- Estruturação com HTML semântico
- Estilização moderna com CSS
- Organização de layout (grid e flexbox)
- Interação com JavaScript
- Uso de `localStorage` para salvar preferências do usuário

---

## 🖼️ Layout (Inspiração)

O design do projeto foi inspirado no Google Keep:

- Cards em grade
- Visual limpo e leve
- Botões minimalistas
- Layout responsivo

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Font Awesome** (ícones)
